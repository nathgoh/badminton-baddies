## 3. Google Sheets caching (brainstorm → plan → implement)
- Diagnosed 429 quota errors from hitting Sheets on every request
- Designed and implemented a 30s TTL in-process cache on `SheetsAdapter._all_rows`
- Write-invalidation on `_append`, `_update_row`, `delete_session`, `delete_court`
- `_find_row_index` intentionally bypasses cache (needs true row index before writes)
- 5 new tests added; all 37 tests passing
- Bonus bug caught during review: delete methods were missing cache invalidation

## 4. Bug fixes
- **Cascade delete**: deleting a session now also removes all associated courts and signups from Sheets
- **Paid column header**: noted that column K of the `signups` sheet tab needs `paid` typed in manually as a header
- 2 new backend tests for cascade delete; 39 tests total passing

## 5. Court time display
